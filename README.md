# 🥗 Smart Nutrition Deficiency Dashboard

A modern and interactive web application that helps users analyze food items, discover nutrients present, identify missing nutrients, and create a personalized weekly diet planner.

---

## ✨ Features

- ✅ Food Nutrition Analysis
- ✅ Missing Nutrient Detection
- ✅ Calories & Weight Display
- ✅ Browse Foods by Categories
- ✅ Weekly Diet Planner
- ✅ Auto Meal Plan Generator
- ✅ Personalized User Profile
- ✅ Responsive Dashboard UI
- ✅ API Integration with Fallback Support

---

## 🛠️ Tech Stack

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript
- 🟢 Node.js
- 🚂 Express.js
- 🍎 Edamam Nutrition API

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/ShahiSameer007/Smart-Nutrition-Deficiency-Dashboard.git

### 2️⃣ Open Project Folder

cd Smart-Nutrition-Deficiency-Dashboard

### 3️⃣ Install Dependencies

> 📌 The node_modules folder is not uploaded to GitHub because it is auto-generated and too large.

npm install

This command will automatically recreate the node_modules folder.

---

## ▶️ Run Project

node server.js

Then open your browser:

http://localhost:3000

---

## 🧑‍💻 How to Use

### 🔍 Analyze Food

Enter food items like:

2 eggs  
1 apple  
1 cup spinach  
200g grilled chicken breast

Then click **Analyze**.

---

### 📂 Browse by Category

Use the Browse feature to explore foods by category:

- 💪 High Protein
- 🩸 High Iron
- 🦴 High Calcium
- 🥦 Vegetarian
- 🍗 Non-Veg
- 🩺 Diabetic Friendly

---

### 📅 Weekly Planner

- Add foods to a specific day
- Add foods to the entire week
- Auto-generate meal plans

---

### 👤 User Profile

Customize:

- Age
- Gender
- Height
- Weight
- Dietary Preference
